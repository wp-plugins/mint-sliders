A wordpress image slider plugin.

== Description ==

This is a wordpress images slider plugin. nextgen-gallery plugin is used to manage and upload images. skitter transaction effects are used in frontend.
nextgen-gallery is a required plugin. Please download and install from http://wordpress.org/extend/plugins/nextgen-gallery/. And do not forget to activate it after installation.
[Demo](http://taraprasad.com/wordpress/mintsliders/ "Demo")

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin archive file.
2. Expand it to wp-content/plugins/ directory or can be uploaded the archive file from wordpress backed.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Download nextgen-gallery plugin and install from http://wordpress.org/extend/plugins/nextgen-gallery/
5. Activate the nextgen-gallery plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How to upload images? =

Create a new gallery using nextgen-gallery and upload images.

= How to select images for slider? =

Create a new image slider and select the gallery created in nextgen-gallery from dropdown.

= Where is shortcode? =

Shortcodes are in the Image sliders listing page. Just copy and paste anywhere you want.
